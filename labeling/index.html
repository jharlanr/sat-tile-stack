Labeling Guidelines

✕

Assign probability estimates for each drainage class. Probabilities must sum to 1.0. The label is automatically set to the class with the highest probability (argmax). Use probability splits (e.g., 0.50/0.50) when uncertain. Flag ambiguous cases for revisit.

ND — No Drainage

  • Lake does not drain its water in any way (laterally or englacially)
  • If there is any outlet stream, even a small one, it is not ND — that would be LD
  • A lake can still be ND even if streams are visible — if those streams flow into the lake (inlets), not out of it
  • To distinguish inlet vs outlet: streams generally taper from wide to narrow in the direction of flow, but this can be tricky — check whether the lake level drops over time (outlet/drainage) or stays stable/rises (inlet only, likely ND)

HF — Hydrofracture

  • Rapid drainage (often ~1 day) where water drains within the lake extent
  • Look for: fracture scarps, ice chunks, sudden area decrease
  • If a lake drains into a moulin AND also hydrofractures → label as HF
  • New moulins that form from hydrofracture → still HF
  • If truly unclear moulin vs HF → label as HF and flag for revisit

MD — Moulin Drainage

  • Drainage into a moulin (vertical channel into the ice)
  • Tends to be slower than hydrofracture
  • Moulin is often outside the lake extent — don't confuse with lateral drainage
  • Moulin must be visible within the 512×512 frame (water disappears into ice in-frame)
  • Moulins sometimes become more obvious later in season after snow falls
  • If a lake drains via outlet stream that terminates in a moulin → label as MD
  • If a lake drains via stream AND separately into a moulin → label as MD

LD — Lateral Drainage

  • Lake drains via an outlet stream or slush field
  • If the outlet stream meanders out of frame (even if it might end in a moulin) → LD
  • Lake-to-lake drainage is always LD — if a lake drains into another lake's basin, even if that neighboring lake drains via HF or moulin, the source lake is LD
  • If a lake sits in a crevasse field but clearly drains via outlet stream → LD, not CD

CD — Crevasse Drainage

  • Lake drains into a crevasse field
  • Catastrophic HF-like drainage in a crevasse field → still CD
  • Lake just outside/touching a crevasse field that drains → CD
  • Lake drains via stream into a crevasse field → CD
  • Lake in crevasse field but clearly drains via outlet stream out of frame → LD, not CD
